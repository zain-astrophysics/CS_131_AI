# Behavior Tree of a vacuum cleaner

** Description**
This Python project uses reflex agent to understand its environment and respond correspondingly.

** Package Information **

1. bt: The "bt" folder consists of sub-folders such as composites, conditions, decorators, tasks
	It also consists of three scripts 
		a. -init_.py that initializes the bt
		b. behavior_tree.py contains the actual algorithm of tree
		c. global.py contains the initialized variables that will be used in scripts
2. bt_library: It consists of useful libraries that will be used throughout the package
3. main.py: Runs the whole code


*** Running the Python script ***
1. The user should open the main.py and run the code

2. The user will be prompted to select one of the two choices: 
   1. Base version, 2. Experimental version

3. The code will run and proceed as instructed.
